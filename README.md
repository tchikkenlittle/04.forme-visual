# FORME Visual — Portfolio Website

Demo de site de portfólio para fotógrafo de produto e e-commerce, focado
em moda e apparel DTC. Construído com HTML, CSS e JavaScript puro.

## 🎯 Objetivo
Demonstrar um site clean, minimalista e responsivo para o nicho de
fotografia de moda e vestuário. Projeto desenvolvido para portfólio de
vendas no Fiverr.

## 🛠️ Tecnologias
- HTML5 semântico
- CSS3 (variáveis, flexbox, grid, animações)
- JavaScript vanilla (Intersection Observer, menu mobile, fetch API)
- Formspree (backend do formulário de contato)
- Google Fonts (Cormorant Garamond + Inter)
- Imagens via Unsplash

## 📁 Estrutura do Projeto
- `index.html` — arquivo único com todo o site
- `README.md` — este arquivo
